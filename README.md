## ✨ À propos

Velkora Client est un launcher Minecraft développé avec Electron, conçu pour simplifier l'accès au jeu tout en proposant une expérience moderne et intuitive.

Pensé pour les joueurs souhaitant une solution complète, Velkora Client intègre la gestion des comptes Microsoft, des profils personnalisés, des mods et diverses optimisations pour rendre le lancement du jeu simple et rapide.

## 👨‍💻 Développement

Cloner le projet :

```bash
git clone https://github.com/pharos-off/Velkora-Client.git
cd Velkora-Client
```

Installer les dépendances :

```bash
npm install
```

Lancer en mode développement :

```bash
npm run dev
```

Construire l'application :

```bash
npm run build
```
